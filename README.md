# clickWave.js

__________________

点击波浪效果插件。  
对用户的点击交互行为，有良好的视觉反馈。

__________________

## USAGE 使用方法

直接收入 `clickWave.js` 即可！

```html
<!DOCTYPE html>
<html>

<head>
  <meta charset="utf-8">
</head>

<body>
  在页面上任意地方点击，即可看到效果！
  <script src="../clickWave.js"></script>
</body>

</html>
```

**[demo 查看演示](https://cdn.rawgit.com/wusfen/clickWave.js/master/example/demo.html)**
